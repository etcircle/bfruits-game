# Roadmap

## Implemented Features

### Player Movement
- [x] WASD Movement (Camera Relative)
- [x] Jumping & Double Jump
- [x] Sprinting
- [x] Ground Dash & Air Dash
- [x] Sliding
- [x] Gliding

### Combat
- [x] Basic Attack (Left Click)
- [x] Hit Detection (Cone-based)
- [x] Enemy Health & Damage

### Enemies
- [x] Basic AI State Machine (Idle, Chase, Attack)
- [x] Melee Enemies
- [x] Ranged Enemies (Projectiles)
- [x] Health Bars & Visual Indicators

### World
- [x] Basic City Layout
- [x] Building Types (House, Shop, Tower, Temple)
- [x] Environmental Props (Trees, Fountain)

## Planned Features

### Short Term
- [ ] **Wall Running**: Mechanics for traversing vertical surfaces.
- [ ] **Combat Polish**: Better visual effects for hits and attacks.
- [ ] **Enemy Variety**: More enemy types and behaviors.

### Medium Term
- [ ] **Quest System**: NPC interactions and objectives.
- [ ] **Inventory**: Loot and equipment management.
- [ ] **Skill Tree**: Unlockable abilities and upgrades.

### Long Term
- [ ] **Multiplayer**: Networking and replication.
- [ ] **Boss Fights**: Complex encounters with unique mechanics.
- [ ] **Save/Load**: Persistence for player progress.
